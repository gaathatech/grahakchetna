## ðŸ”§ Technical Details

### Layout Parameters Saved
When you generate a video, these layout settings are stored:
- Media Position (left/right/center)
- Media Size (35-100%)
- Media Opacity (0-100%)
- Text Alignment (left/center/right)
- Background Blur Level (none/light/medium/heavy)

### Video Output
- Format: MP4 (H.264 video + AAC audio)
- Long form: 1920x1080 (16:9)
- 24 FPS
- Stored in: `videos/long/` folder
- Added to manifest/archive automatically

## â“ FAQ

**Q: Can I change the layout after generation?**  
A: No, layout is applied during generation. Re-generate with new settings if needed.

**Q: Does layout affect video quality?**  
A: No, all videos are generated at full 1920x1080 resolution.

**Q: Which preset should I use?**  
A: Start with "Default" - it's optimized for most content types.

**Q: Can I preview the layout before generation?**  
A: Yes! The preview shows exactly how your layout will look.

**Q: Does media position affect rendering time?**  
A: No, rendering time is the same regardless of layout.
